# Git指令一览

## 常用指令
```
git init            初始化仓库
git clone           拷贝一份远程仓库，也就是下载一个项目。

git add             添加文件到暂存区
git status          查看仓库当前的状态，显示有变更的文件。
git diff            比较文件的不同，即暂存区和工作区的差异。
git commit          提交暂存区到本地仓库。[可选参数 -m "message"]

git pull            下载远程代码并合并
git push            上传远程代码并合并
```
